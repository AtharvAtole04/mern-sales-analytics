# Sales Analytics Dashboard

A full-stack business analytics dashboard built using the MERN stack.

This project helps businesses manage products, monitor inventory, track revenue, and analyze sales performance using interactive dashboards and visual reports.

---

## Features

### Product Management
- Add Products
- View Products
- Update Products
- Delete Products

### Data Analytics
- Total Products Overview
- Total Inventory Quantity
- Revenue Calculation
- Sales Visualization Charts

### Data Handling
- Search Products
- Sort Products by Price
- Export Data as CSV

### UI Features
- Dashboard Cards
- Product Table
- Responsive Layout
- Interactive Charts

---

## Tech Stack

### Frontend
- React.js
- Axios
- Chart.js
- React CSV

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## Installation

### Clone Repository

```bash
git clone <your-repository-url>
```

---

## Backend Setup

```bash
cd backend
npm install
npm run dev
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm start
```

---

## Environment Variables

Create a `.env` file inside backend:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
```
